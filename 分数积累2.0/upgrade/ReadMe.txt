通常，人们对没有立刻回报的付出不会有兴趣，可在现实中，很多付出都不会有立刻的回报。
头衔是一种虚荣，人都是有虚荣心的。
学习如逆水行舟，不进则退，科学的计分也应当如此。
学习需要专注，不集中注意力不可能有高效率。人的精力是个常数，人注意的事情是个变量，效果=精力/注意的事情的数量。
一个人应该对自己的水平有准确的认识，人贵自知。

规则1:
一个小时积累一分。每积分到一定数量就升级，换头衔。

规则2:
每此思想开小差超过5分钟，当次积分减去百分之十。

规则3:
将所有积分除以50，余数为浮动分数（从总分中分离出来的），若每天积分不足2分，则浮动分数减去2分再加入总分中，直到浮动分数减为0。

规则4:
设置单日最高分，自己挑战自己，不断自我超越。
