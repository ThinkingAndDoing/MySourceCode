1. install pywin32 module

1).go to https://sourceforge.net/projects/pywin32/files/pywin32/Build%20220/ download the files you need.
   if you install win32 python ,you need to download pywin32 ,And the pywin32 version should be the same as the python version you install.
2).install step by step.
3).ensure install success-> python ->import win32com

2. install future module
1). unzip future-0.16.0.tar.gz
2). enter unzipped folder,run:python setup.py install
3). ensure install success-> python->import future


3. install pefile module
1). unzip pefile-master.zip
2). enter unzipped folder,run :python setup.py install
3). ensure install success-> python -> import pefile

4. install pyinstaller 
1). unzip PyInstaller-3.2.zip
2). enter unzipped folder,run python setup.py install
3). ensure install success -> pyinstaller --version

usage:
pyinstaller -F -w test.py
